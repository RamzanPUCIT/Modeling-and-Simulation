# Modeling and Simulation

This repository contains my coursework and practical implementations for the **Modeling and Simulation** subject (M.Phil AI). It focuses on building a strong foundation in computational modeling, simulation techniques, and system analysis.

---

## 📘 Overview

Modeling and Simulation is about representing real-world systems using mathematical and computational models, and then experimenting with them using simulations to analyze behavior, predict outcomes, and support decision-making.

This repository covers **pre-mid semester concepts** with practical Python implementations.

## 🧠 Topics Covered

### 🔹 1. Basics of Modeling
- System vs Model
- Static vs Dynamic systems
- Discrete vs Continuous systems
- Modeling process (problem → model → validation → analysis)

---

### 🔹 2. Linear Models
- Equation of line from two points
- System of linear equations
- Matrix representation
- Demand and supply equilibrium
- Concept of linearity

---

### 🔹 3. Random Numbers & Distributions
- Pseudo-random numbers
- Linear Congruential Generator (LCG)

### 🔹 4. Monte Carlo Methods
- Random sampling techniques
- Estimation of π

### 🔹 5. Queuing Theory
- M/M/1 queue model

### 🔹 6. Markov Chains

- Equilibrium (steady-state) concept

### 🔹 7. System Dynamics (Continuous Models)

- Logistic growth models

- Real-world models (population, inventory, economics)
